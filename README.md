# Orbline
### is a orbis style outlining tool
![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)
## Features
- simple ui for editing outlines
- programmatic  spreadsheet generator
- powerful shortcuts

# install & development
```bash
pnpm install
pnpm run dev
```
